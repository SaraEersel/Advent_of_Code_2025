# Advent of Code 2025

## What is Advent of Code?
Advent of Code is an annual coding challenge that runs during December. Each day, a new problem is released consisting of two parts. The puzzles are designed to be fun, educational, and sometimes quite challenging.  
Check it out [here](https://adventofcode.com/).

## Repository Purpose
This repository contains my solutions for Advent of Code 2025. The goals are:
- To improve problem-solving skills.
- To learn new programming languages/tools.
- To document the thought process and solutions.
